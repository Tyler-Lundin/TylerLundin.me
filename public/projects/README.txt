Drop project screenshots and short video loops here.

Recommended:
- Tall full-page screenshots (PNG/JPG/WebP) for auto vertical pan
- Short MP4/WebM loops for motion demos (muted, looped)

Usage:
- Reference in data: src/data/projects.ts
- Example: src: "/projects/my-site-hero.png"
- To force/disable auto pan, set media.autoScroll = true|false
- Adjust speed via media.scrollDurationMs (default ~16000ms)
