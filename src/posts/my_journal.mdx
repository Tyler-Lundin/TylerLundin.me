MyJournal is a React and Firebase-powered journaling website that was the first project I undertook as a self-taught web developer. While building it, I learned many valuable lessons about managing application state effectively.

At first, I used the `useState` hook provided by React, but quickly found it was becoming unwieldy as the project grew in complexity. To tackle this challenge, I decided to overhaul the state management system and learn to use Redux and the Redux Toolkit, which helped me keep the code cleaner and more organized.

As a result of this experience, I gained a much deeper understanding of React and its associated libraries, as well as the importance of effective state management in larger projects. MyJournal reflects my journey as a developer, and I am excited to continue expanding my knowledge and skill set with future projects.
